Formatações!

Código Fonte:

Para ajeitar o espaçamento entre as letras do seu código fonte use a tag <code>.

EX: O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Com a tag <code> ficará assim: O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Tag <pre>

Mostra na tela o que você escreveu da mesma maneira em que está no editor de texto, no meu caso no Visual Sdudio Code. EX:

Se estiver assim: 

    <pre>
        Gui
           lherme
                 João
    </pre>

Ira aparecer assim : 

            Gui
               lherme
                     João

Citações Simples!

Use a tag <q> EX: Disse ele: Guilherme é um jogador espetacular!

Citações Completas!

Se você quer usar uma citação longa de determinado autor use a tag <blockquode>.Vai deslocar seu texto para direita, EX:

"O Felipão é um líder do seu povo, os dá moradia! Em troca pagam impostos e obedecem a ele sem reclamar! Uma vez desobedeceram ele e não demorou muito para o arrependimento vir, foi PÁ PUM! KKKKK."

Dentro da tag <blockquote> dá para se usar um parâmetro, seria ele o: cite="". Com esse parâmetro você cita o livro de algum autor por exemplo! Você pode encontrar diversos livros no Google Livros. Você pega o link e coloca esse link no parâmetro cite= "". Assim você mostra de onde veio o conteúdo e dá o devido crédito ao autor! Visualmente não ira aparecer nada mas é importante para vendas e outras coisas mais!

A citação abaixo está citando um trecho longo do livro de alguem, achado lá no Google Livros!

"Honestamente, acho que ter jogado o primeiro ajudou... Ainda foi preciso um empurrãozinho aqui e ali, mas depois de ler tantos romances e jogar o segundo jogo, Geralt já estava na minha cabeça."

Abreviações!

HTML é uma abreviação. Use a tag <abbr> e explique o que essa abreviação significa, quando for passado o mouse em cima dessa abreviação ira aparecer o significado que você inserir dentro da tag!
