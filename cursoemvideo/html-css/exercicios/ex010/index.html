Links no Seu Site!

Para se por um link você precisa usar a tag <a>, com ela virá o parâmetro: href="". Significa: Referência hiper texto, no meio das aspas você colocará uma URL, um link! HAHAHAHAHA

Mas esse link tem um problema... Ele abre em "cima" do seu site. Sempre que você quiser levar alguém para outra página use os parâmetros: target="_blank" e o rel="external".

Target="_blank" = Janela em branco, nova guia.
Rel="external" = link extermo.
