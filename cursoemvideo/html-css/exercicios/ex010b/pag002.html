Essa é a segunda página, se deseja criar um link para voltar para a primeira use a tag <a> </a>. No atributo href="", pressione ctrl + espaço e selecione a página anterior.Dentro da tag <a>, use o atributo rel="", no meio dessas aspas digite prev.

Esse atributo auxilia o mecanismo de busca, "diz" para ele que o link é para a parte anterior do documento atual.
