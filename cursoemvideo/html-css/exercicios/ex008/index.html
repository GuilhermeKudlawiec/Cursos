Marca texto!

Tag <mark> faz isso: SHIRA.

Trocando cor da tag <mark>.

Para trocar a cor use CSS! Digite style na frente de mark e precione ENTER em style, na frente de style selecione background-color, escolha a cor e fim!

EX:Tag <mark> faz isso: SHIRA.

Se exixte uma maneira de mudar a cor de diversos marks de uma só vez! É inserindo <style> em <head!>

Texto Grande e Texto Pequeno!

Para grande temos a tag <big>, ela esta obsoleta mas funciona.

Para pequena se tem a tag <small>, não está obsoleta porque possuí semântica, se tem um significado! EX: Olha ela funcionando!

Texto Deletado e Inserido.

Podemos marcar um texto como excluído, indicar que ele deve ser lido mas não considerado. Isso com a tag <del>.

Podemos marcar um texto como inserido, para dar ênfase e indicar que ele foi adicionada depois. Isso com a tag <ins>.

Existe também o sublinhado com a tag <u> (não semântica).

Texto Sobrescrito.

Para inserir coisas do tipo: x20 + x10. Isso com a tag <sup>.

Texto Subscrito.

Para inserir coisas do tipo: H2O. Isso com a tag <sub>.
