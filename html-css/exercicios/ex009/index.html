Listas em HTML

Listas Ordenadas

A ordem importa, use a tag: <ol>. Cada item da lista deve estar dentro da tag: <li>. Veja o exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent

Mude a formatação com o parâmetro type="". Veja os exemplos:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent

Você pode começar com a ordem que você quiser! Isso com parâmetro start="". Veja o exemplo, ele vai na frente do type!

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Arrumar a cama
  5. Jogar Gwent

Listas Não Ordenadas

Para elas use a tag: <ul>. E não esqueça de colocar os itens da lista dentro da tag: <li>. EX:

  • God of War
  • The Last of Us Part II
  • Red Dead Redemption II
  • Crash Bandicoot
  • The Witcher 3
  • Gwent
  • Uncharted
  • Horizon Zero Dawn
  • Sekiro
  • Bloodborne
  • Dragon Ball FigtherZ
