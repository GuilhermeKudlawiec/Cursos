Links Internos!

São links que irão abrir uma nova página, página essa que esta no nosso servidor. A tag é essa: <a> </a>.

Para se ter links internos precisamos ter mais de uma página, no Visual Sdudio Code siga os seguintes passos: Explorador → Novo Arquivo → Nomeie seu arquivo (.html no final!) → Enter.

Na nova página desenvolva ela da mesma forma que essa, crie a estrutura do html e "meta bala".

Para acessar a página criada use a tag <a> </a>. No atributo href="" pressione ctrl + espaço e selecione a página criada. Dentro da tag <a>, use o atributo rel="", no meio dessas aspas digite next. Indica que o link é para a próxima parte do documento atual. Existe também o atributo target="_self", significa: abrir o link na janela ou frame atual (padrão).
